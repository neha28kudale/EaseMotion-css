Replace Hardcoded outline-offset with CSS Variable

Issue #12200 — components/buttons.css hardcodes outline-offset: 3px instead of using a CSS variable.

Demo: Focus-Visible Buttons

Tab through the buttons below to see the focus outline-offset controlled by --ease-outline-offset.

Demonstration of CSS Variable Override

The following section uses a custom --ease-outline-offset: 8px to show global override works:

How This Fix Works

/* Before (hardcoded in components/buttons.css) */
.ease-btn:focus-visible {
  outline-offset: 3px;
}

/* After (uses CSS variable with fallback) */
:root {
  --ease-outline-offset: 3px;
}

.ease-btn:focus-visible,
.ease-btn-link:focus-visible {
  outline-offset: var(--ease-outline-offset, 3px);
}

Note: No core files were modified. This submission proposes the change by demonstrating the CSS variable approach.
